# Sales OCR

Sales OCR is an innovative application designed to streamline the process of managing sales transactions by leveraging Optical Character Recognition (OCR) technology. This application allows users to upload sales receipts, extract relevant information using OCR, and automatically create sales transactions in the system. It also synchronizes inventory and recognizes customer data for a seamless experience.

## Features

- Upload sales receipts
- OCR text extraction
- Automatic sales transaction creation
- Inventory synchronization
- Customer data recognition

## Tech Stack

- Backend: Node.js, PostgreSQL, Tesseract OCR / Google Vision AI
- Frontend: Next.js / React, TypeScript, Tailwind CSS
- Deployment: Render
